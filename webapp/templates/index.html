About PlayMeow

PlayMeow is an AI-powered interactive laser toy designed to maximize cat engagement through machine learning. The system:

  • Analyzes cat behavior and movement patterns
  • Learns to predict movement that encourages play
  • Uses reinforcement learning to improve engagement over time
  • Provides a simulation environment for testing
  • Can be deployed to a Unity-based hardware application

Start by training the model, or run a simulation to see the current model in action.
